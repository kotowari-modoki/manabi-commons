# 図工・アートで楽しく表現しよう

図工は、学校だけで終わりではありません。

- 見て、感じて、表現すること
- 手を動かして何かを作ること
- 失敗しながら工夫すること

は、家でも続けられます。

このセクションでは、小学生が家でできるアート活動と、保護者のみなさんが安心して準備できるガイドをまとめています。
