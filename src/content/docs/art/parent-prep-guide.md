# 保護者のための準備ガイド

「絵の具を出すと汚れそう」「何を用意すればいいかわからない」という保護者のかたに向けて、
おうちアートを気軽に始めるためのヒントをまとめました。

---

## 最初に揃えておきたい道具

まずはこれだけあれば、ほとんどの活動を楽しめます。

| 道具 | 目安・選び方 |
|------|-------------|
| 画用紙（A4〜B4） | 100円ショップの厚口タイプで十分 |
| スケッチブック | 薄めでOK。鉛筆スケッチ用 |
| 水彩絵の具 | 学校指定品か12色セット |
| 鉛筆（B・2B） | 低学年は2B、高学年はBが描きやすい |
| 色鉛筆 | 12〜24色。水性より油性の方が発色がよい |
| 太めの筆・細めの筆 | 各1本ずつあると使い分けられる |
| ふた付きの水入れ | 倒れにくいもの。食品用タッパーでも代用可 |
| 絵の具用パレット | 仕切り付き。紙皿でも代用可 |
| 新聞紙・レジャーシート | 机や床を保護するために必須 |
| 使い古しのタオル・ウエス | 手や筆を拭くため |

---

## 場所を整えるコツ

### 汚れを気にしない環境をつくる

- **新聞紙を敷く** ── 机の上に2〜3枚重ねて敷くだけで、後片付けが格段に楽になります
- **レジャーシートを活用** ── 床に敷けば絵の具が落ちても安心
- **古着を作業着に** ── お気に入りの服では集中できないので、汚れてもいい服を「アート用」として決めておくと子どもも気が楽です

### 作業スペースの広さ

- 最低でも A3 サイズ（A4 の2倍）くらいの広さが確保できると動きやすいです
- 床に座って大きな紙に描くスタイルも、低学年には向いています

### 乾かす場所を決めておく

- 完成した作品は乾かす必要があります。洗濯物干しやクリップで壁に貼れる場所を決めておくとスムーズです

## 道具の後片付け

子どもが「また使いたい」と思えるかは、片付けの気持ちよさにかかっています。

**筆の洗い方**（毎回大切に）

1. 水入れで軽くすすぐ
2. 石けんを少しつけて、手のひらで優しくこねる
3. 根元まで洗って、水でよく流す
4. ティッシュや布で水を押さえて、穂先を整えて乾かす

**絵の具パレットの扱い**

- 乾く前に水でさっと流せば簡単に落ちます
- すでに固まった場合は、水に浸けて数分待ってから洗いましょう

## 安全に使える素材の目安

| 素材 | 注意点 |
|------|--------|
| 水彩絵の具 | 体につくが水で落ちる。アレルギーがあるお子さんは確認を |
| 油性マーカー | 換気を忘れずに。服につくと落ちにくい |
| のり（でんぷん系） | 安全。乾いたらきれいに落とせる |
| はさみ | 低学年には丸刃タイプを。使い方は都度確認を |
| 竹串・つまようじ | スクラッチ画で使用。使う際は親が近くにいてください |
| 石こう・粘土 | 乾燥するので密閉保存。切れ端は燃えるごみへ |

## 子どもへの声かけのコツ

### やってほしいこと

- **「どんなこと考えて描いたの？」** と聞く ── 作品の出来より、気持ちと過程を大切に
- **「ここ、面白いね」** と具体的にほめる ── 「上手」より「この色の組み合わせが好き」のほうが伝わる
- 完成しなくてもOKと伝える ── 途中でやめてもいい。また続けていい

### 避けてほしいこと

- 「もっとうまく描けるよ」という比較や修正の提案
- 「早く終わらせなさい」という時間のプレッシャー
- 「汚すから嫌だ」という発言 ── 汚れることへの罪悪感が、表現の萎縮につながります

## 作品の保管アイデア

| 方法 | 特徴 |
|------|------|
| クリアファイルに入れて保管 | 手軽。A4・A3サイズをそろえておくと便利 |
| 写真に撮ってデジタル保存 | かさばらない。スマホで十分 |
| 1枚だけ額に入れて飾る | 特別な作品を選ぶ体験になる |
| 学期ごとにまとめてポートフォリオに | 成長の記録になる |

## 保護者のかたへ

アート活動で大事なのは「正解を出すこと」ではなく、**「何かを感じて、手を動かすこと」** です。

お子さんが机に向かって無心で絵を描いているとき、それはとても豊かな時間です。

保護者のかたの役割は、その時間を守ること。
道具を整え、汚れを気にしない空間をつくり、終わったあとに「楽しかった？」と聞いてあげてください。

それだけで、子どもの表現はどんどん広がっていきます。
