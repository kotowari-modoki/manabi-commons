# 小学生の算数で学ぶこと

このページでは、小学校の6年間で学ぶ算数を、できるだけわかりやすくまとめています。

算数は、ただ答えを出す勉強ではありません。

- 数や形に気づくこと
- どうしてそうなるのか考えること
- 図や式やグラフで表すこと
- くらしの中で使うこと

を大切にして学んでいきます。

## 算数ってどんな勉強？

算数では、こんなことを少しずつ学びます。

- 数を数えること
- たし算、ひき算、かけ算、わり算
- 長さやかさや重さ、時間
- 三角形や四角形、円などの形
- 表やグラフの見方
- きまりを見つけて考えること

学年が上がるほど、もっと大きな数や、もっとくわしい考え方を学ぶようになります。

## 学年ごとの学び

1年生から6年生まで、こんな順番で学んでいきます。

### 1年生

1年生では、算数のはじめの土台を作ります。

- 数を数える
- たし算とひき算をする
- いろいろな形を見つける
- 長さや広さをくらべる
- 時計を読んで時こくを知る

「いくつあるかな」「どちらが大きいかな」と考える力を育てます。

### 2年生

2年生では、数や計算の力をもう少し広げます。

- 大きい数を読む
- たし算とひき算をもっとたしかにする
- かけ算九九をおぼえる
- 長さやかさをはかる
- 時間を考える
- かんたんな表やグラフを見る

「同じ数ずつある」ときに、かけ算で考えられるようになります。

### 3年生

3年生では、新しい数の表し方や計算が増えます。

- わり算を学ぶ
- 小数を知る
- 分数を知る
- 三角形や円について学ぶ
- 重さや時間をくわしく考える
- 棒グラフを読む

ここから、数の世界がぐんと広がります。

### 4年生

4年生では、より大きい数や、少し複雑な計算を学びます。

- とても大きい数を読む
- 四捨五入やおよその数を使う
- わり算をもっとくわしく学ぶ
- 小数や分数の計算を進める
- 面積や角を学ぶ
- 折れ線グラフを見る

「だいたいどれくらいかな」と見当をつける力も大切になります。

### 5年生

5年生では、しくみや関係に目を向けて考えることが増えます。

- 小数のかけ算とわり算
- 分数のたし算とひき算
- 割合や百分率
- 速さ
- 合同な図形
- 面積や体積
- 円グラフや帯グラフ

くらしの中の数字を、より正しく読み取る力を育てます。

### 6年生

6年生では、小学校の算数のまとめをします。

- 分数のかけ算とわり算
- 文字を使った式
- 拡大図と縮図
- 線対称や点対称
- 円の面積
- 柱の体積
- 比例、反比例、比
- データの見方やくらべ方

中学校の数学につながる大切な学びがそろっています。

## 算数で大切にしていること

算数では、答えが合っていることだけでなく、

- どう考えたかを説明すること
- 図や式を使って伝えること
- 友だちの考えを聞いてくらべること
- 学んだことを生活に生かすこと

も大切にしています。

「わかった」で終わらず、「使える」ようになることが目標です。

## まとめ

小学校の算数は、

- 数
- 計算
- 形
- はかること
- 表やグラフ

を学びながら、考える力と伝える力を育てる勉強です。

1年生から6年生まで、少しずつできることを増やしていきます。
